# EEG_CUDA：高性能脑电信号处理加速器

[ 英文 | [中文](./README_zh.md) ]

EEG_CUDA 是一个专为脑电图（EEG）信号处理设计的高性能库。通过利用 NVIDIA CUDA，本项目在保持高数值稳定性的同时，加速了用于长序列、多通道数据的关键算法。

数学细节请参阅 [math.md](docs/math.md)。

## 环境要求

- Python 3.11
- PyTorch 2.6.0+cu124（需搭配 CUDA 12.4）
- CUDA Toolkit 12.4（与 PyTorch 编译版本一致）
- 计算能力不低于 7.0 的 NVIDIA GPU（推荐）

## 🛠 安装与配置
请按照以下步骤配置环境并构建 CUDA 扩展。

1. 克隆仓库
```bash
git clone https://github.com/643591947/eeg_cuda_ops.git
cd eeg_cuda_ops
```
2. 安装依赖
建议使用虚拟环境
```bash
pip install -r requirements.txt
```

3. 构建 CUDA 扩展
```bash
pip install -e . --no-build-isolation
```

4. 运行可视化界面
```bash
cd web_UI
streamlit run main.py
```
