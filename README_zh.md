# EEG_CUDA：高性能脑电信号处理加速器

[ English | [中文](./README_zh.md) ]

EEG_CUDA 是一个专为脑电图（EEG）信号处理设计的高性能库。通过利用 NVIDIA CUDA 技术，本项目在保持高数值稳定性的同时，加速了长序列、多通道数据的关键算法。

数学原理详见 `docs/math.md`。

## 环境要求

- Python 3.11
- PyTorch 2.6.0+cu124（需 CUDA 12.4 支持）
- CUDA Toolkit 12.4（与 PyTorch 编译版本匹配）
- 计算能力 ≥ 7.0 的 NVIDIA GPU（推荐）

本扩展基于以下环境构建并测试：
- PyTorch 2.6.0+cu124
- 构建命令：`pip install -e . --no-build-isolation`
